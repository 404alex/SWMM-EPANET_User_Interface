# Load Example1 and save to a new file

Start SWMM.

Open the file named 'Example1.inp' using 'File:Open'.

Save the project using 'File:Save As..' with the name 'Example1z5.inp'.

Save the project using 'File:Save As..' with the name 'Example1z6.inp'.

Compare files 'Example1z5.inp' with 'Example1z6.inp'. Should be no difference

Click 'OK'.
