# Load Example1 and save to a new file

Open the file named 'Example1.inp' using 'File:Open'.

Save the project using 'File:Save As..' with the name 'Example1z7.inp'.

Save the project using 'File:Save As..' with the name 'Example1z8.inp'.

Compare files 'Example1z7.inp' with 'Example1z8.inp'. Should be no difference
