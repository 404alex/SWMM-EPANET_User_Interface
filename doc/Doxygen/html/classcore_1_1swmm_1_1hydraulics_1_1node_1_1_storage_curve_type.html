Detailed Description

Type of flow divider.

Choices are: CUTOFF (diverts all inflow above a defined cutoff value), OVERFLOW (diverts all inflow above the flow capacity of the non-diverted link), TABULAR (uses a Diversion Curve to express diverted flow as a function of the total inflow), WEIR (uses a weir equation to compute diverted flow).
